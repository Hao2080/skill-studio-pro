# Skill Studio Pro brand placeholder

The assets in `public/assets/brand/` are independent Wave 0 placeholders created for Skill Studio Pro. They do not reuse the upstream Skill Studio logo.

These placeholders may be replaced by approved final Pro brand assets in a later brand-specific workstream. Until then, the Pro application shell, browser metadata, documentation, and generated desktop icons must use these files rather than the upstream logo assets retained in Git history.
