# Upstream logo reference archive

This directory preserves logo files and preview assets imported from the upstream Skill Studio repository at commit `cd0bb0af53865d4a9643968080bfc5a8137b72d9`.

These files are retained for attribution and historical reference only. They are outside `public/`, are not bundled by the Skill Studio Pro frontend, and must not be used as the formal Pro brand. The runnable Pro baseline uses the independent placeholder assets in `public/assets/brand/` and the desktop icons generated from those placeholders.
