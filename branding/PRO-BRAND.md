# Skill Studio Pro brand

The formal Skill Studio Pro identity is defined by the independent assets in `public/assets/brand/`:

- `skill-studio-pro-mark.svg`: an abstract three-layer ocean wave forming an S.
- `skill-studio-pro-wordmark.svg`: the product wordmark used by repository documentation.

The visual identity uses deep ocean blue, cyan highlights, and a restrained indigo secondary accent. It does not reuse Apple or upstream Skill Studio proprietary brand material. Desktop application icons are generated from the same Pro mark so the browser, application shell, and packaged app remain consistent.
